# random_mesh

# Example: 3D-IWGAN
This example lets you train the 3D-IWGAN method. See: https://arxiv.org/abs/1707.09557 for details. 


### Training the network: 

To train run
```
python train.py --modelnet-root path/to/modelnet --cache-dir cache/
```
